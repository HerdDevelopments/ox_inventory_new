# 🎒 Redesigned ox_inventory – Level-Based Slot System

A redesigned version of **ox_inventory** featuring a **progression-based inventory system** where players must upgrade backpacks to unlock more slots.

---

## 📌 Overview

This script introduces a more immersive inventory experience by limiting player slots at the start and allowing expansion through **backpack upgrades**.

Instead of having full access immediately, players must **purchase higher-level backpacks** to increase their carrying capacity.

---

## ✨ Features

* 🎒 **Backpack-Based Slot System**
* 📈 **5 Upgrade Levels** (progressively unlock all slots)
* 💰 Purchase-based progression system
* ⚙️ Clean and optimized redesign
* 🔌 Easy to integrate with existing servers
* 🆓 **Completely Free**

---

## 🎮 How It Works

* Players start with a limited number of inventory slots
* Backpacks can be purchased/upgraded
* Each upgrade increases available slots
* Level 5 backpack unlocks full inventory capacity

This system adds:

* Better economy balancing
* Player progression
* More realistic inventory management

---

## ⚙️ Requirements

* **ox_inventory**

## 📥 Installation

1. Download the script
2. Place it in your `resources` folder
3. Ensure it in your `server.cfg`:

   ```
   ensure your_resource_name
   ```
4. Configure settings as needed
